Company Profile

GrandTix is a ticket selling service specialized in selling tickets for motorsport races.

Due to the current culture of ticket re-selling, finding tickets at retail price has been getting harder and harder. GrandTix is a service aimed to help users purchase retail priced tickets for Motorsports events with a focus on Formula 1.

Customer Profile

To help combat re-selling of tickets, GrandTix will limit purchases to one per user and require I.D.s, additionally tickets will not be handed out days before the event.

The target audiennce for GrandTix would be motorsport fans looking to get a ticket for retail price since most events sell out instantly and tickets are re-sold with a markup

Logo


4th iteration of the logo


Main Font - Titillium Web

This is the main font GrandTix uses, Titllium web. It will be used for lower headings and most of the body text as well as the font for the buttons and navigations

This is the bolded heavy font used for highlighting important info

This is what empahiszed text looks like for disclaimers etc.

Secondary Font - Exo 2

This font is mainly used for Headings or any text that needs emphasis This is what Exo 2 looks like

Colour

RGB(225,0,0)

This is the primary colour - This will be used for any text, buttons to signify interactability and importance as it contrasts well with the white background

RGB(188,0,0)

This is the secondary colour - This colour acts as an accent colour to be used in cases where the primary colour is already in use

RGB(255,0,0)

This is the tertiary colour - This is the background colour to help contrast the primary colour, also easy to use with the white background
